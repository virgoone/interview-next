## 使用建议

1. 面试前重点背诵"核心回答"，确保基础知识点不遗漏
2. 延伸问题结合自身项目经验补充细节，提升可信度
3. 针对目标岗位侧重复习：前端岗位重点看 React 相关章节，AI 应用岗位重点看 AI 基础与进阶章节
4. 回答时遵循"先核心后细节""先结论后原因"的逻辑，贴合面试沟通场景
